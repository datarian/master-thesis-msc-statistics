2.2 Data Set Usage

The complete data is distributed by the CUP committee pre-split into a learning and validation data set. The validation data was set aside and treated as unseen data for the final prediction. Preprocessing and model selection were performed on the learning data set. The learning data was split 80/20 into training and test sets (see Figure 2.2).
